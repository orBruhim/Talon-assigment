
## Talon assigment

Clone the project

```bash
  git clone git@github.com:orBruhim/Talon-assigment.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start json server for data loading

```bash
  json-server npm run js-server
```

Start the client

```bash
  npm run start
```
